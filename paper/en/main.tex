\begin{abstract}
We present \aletheion{}, a decoder-only language model that integrates
a complete epistemic system as trainable neural modules (\texttt{nn.Module}).
Unlike traditional LLMs that produce only vocabulary logits, each token
generated by \aletheion{} produces a full \textbf{epistemic tomography}
with over 30 fields, including aleatoric ($q_1$) and epistemic ($q_2$)
uncertainty, calibrated confidence via geodesic distance on a 5D
Riemannian manifold, intentionality vector, adaptive temperature, and
cognitive state. The system comprises 11 sub-heads organized in 3 tiers
({\raise.17ex\hbox{$\scriptstyle\sim$}}364K parameters, $<$0.3\% of the model),
trained end-to-end with 13 loss functions and progressive annealing.
We implement Continual Learning via EWC (Elastic Weight Consolidation) and
Experience Replay, with scaling configurations from 1M to 640B parameters.
The 354M parameter model trains on a single RTX 4090 (24GB VRAM).
\end{abstract}

\textbf{Keywords:} LLM, computational epistemology, Riemannian manifold,
calibrated uncertainty, continual learning, decoder-only transformer.
